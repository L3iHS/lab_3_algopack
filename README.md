# Лабораторная работа №3 — Алгоритмический мини-пакет

## Описание
Программа реализует структуры данных, функции для вычисления факториала и чисел Фибоначчи, набор классических сортировок, а собственные классы ошибок и генератор тест-кейсов

---

## Пользовательские ошибки

Все кастомные исключения находятся в `src/errors.py`:

- **StackEmptyError** — обращение к пустому стеку  
- **QueueEmptyError** — обращение к пустой очереди  
- **NonIntegerArgumentError** — аргумент должен быть целым числом  
- **NegativeArgumentError** — аргумент не может быть отрицательным  
- **NonNumericElementError** — передан нечисловой аргумент  
- **InvalidParameterError** — некорректный параметр функции

---

## Факториал и Фибоначчи

Папка `src/fact_fibo/` содержит:

- `factorial` — обычный  
- `factorial_recursive` — рекурсивный  
- `fibo_1` — динамический  
- `fibo_2` — вторая версия  
- `fibo_recursive` — рекурсивная  

Во всех функциях проверяются условия:

- аргумент должен быть `int`, иначе `NonIntegerArgumentError`;
- аргумент `≥ 0`, иначе `NegativeArgumentError`

## Структуры данных

### Стек (`structures/stack.py`)

#### `ListStack`
Стек на списке с функцией `min()` за O(1) \
Использует второй список `_mins` для хранения истории минимумов

#### `LinkedListStack`
Стек на связанном списке \
Каждый узел `_Node` содержит значение и текущий минимум

---

### Очередь (`structures/queue.py`)

#### `ListQueue`
Очередь на списке 

#### `LinkedListQueue`
Очередь на односвязном списке

---

## Сортировки

- `bubble_sort` - сортировка пузырьком
- `quick_sort` - быстрая сортировка
- `counting_sort` - подсчёт элементов (только `int`, смещение для отрицательных чисел)
- `radix_sort(base=10)` - поразрядная сортировка (`int >= 0`)
- `bucket_sort` - сортировка по интервалам
- `heap_sort` - пирамидальная сортировка

---

## Генераторы массивов

Файл `src/generators.py`

### `rand_int_array(n, lo, hi, distinct=False, seed=None)`
Случайный массив целых чисел

### `nearly_sorted(n, swaps, seed=None)`
Почти отсортированный массив

### `many_duplicates(n, k_unique=5, seed=None)`
Массив с малым числом уникальных значений

### `reverse_sorted(n)`
Обратно отсортированный массив

### `rand_float_array(n, lo=0.0, hi=1.0, seed=None)`
Случайные вещественные числа

Во всех функциях используется `seed` для повторяемости результатов
